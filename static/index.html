⚠️ Setup Instructions

  1. Go to Meta for Developers → Your App → WhatsApp → Configuration
  2. Set the Callback URL to your webhook URL above
  3. Set the Verify Token to the value shown above
  4. Subscribe to messages webhook field
  5. Send an image of any invoice to the connected WhatsApp number — YantrAI will auto-parse it!
